Jinja is a template engine for Python.
- is a library designed to combine templates with a data model to produce documents.
- define static parts and dynamic parts (data).

Jinja installation
- $ sudo pip3 install jinja2

Jinja delimiters
- {% %} : statements
- {{ }} : expressions to print to the template output
- {# #} : comments which are not included in the template output
- #  ## : line statements

on the template we can work with:
- dynamic /static var
- objects
- d0ictionaries
